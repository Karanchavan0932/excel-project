# Excel Sales Dashboard Project

## Project Overview
This project analyzes sales data and presents insights using an interactive Excel dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning

## Key Insights
- Sales trends over months
- Orders by gender
- Top performing states
- Order status distribution

## Files
- Excel Dashboard (.xlsx)
- Dashboard Screenshot

## Outcome
Built a dynamic dashboard to help understand business performance and decision-making.
